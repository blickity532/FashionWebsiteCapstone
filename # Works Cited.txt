# Works Cited

## Frameworks and Libraries
1. **Bootstrap**  
   - Source: [https://getbootstrap.com/](https://getbootstrap.com/)  
   - Description: A responsive front-end framework used for styling and layout.

2. **Node.js**  
   - Source: [https://nodejs.org/](https://nodejs.org/)  
   - Description: A JavaScript runtime used for building the backend server.

3. **Express.js**  
   - Source: [https://expressjs.com/](https://expressjs.com/)  
   - Description: A web application framework for Node.js used to create API endpoints.

4. **Font Awesome**  
   - Source: [https://fontawesome.com/](https://fontawesome.com/)  
   - Description: Icon library used for visual elements like the cart icon.

5. **Fetch API**  
   - Source: [https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API](https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API)  
   - Description: JavaScript API used for making HTTP requests to the backend.
